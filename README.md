# Reaction Kingdom

É um jogo de reflexos e raciocínio rápido.  
O objetivo é responder corretamente a cada ronda no menor tempo possível.

### Modos de Jogo

- **Single Player** – 10 rondas.
- **Split Screen** – Dois jogadores no mesmo computador, cada um com os seus controles.

## Como correr
### Passos

1. **Clonar o repositório**
   ```bash
   git clone https://github.com/teu-utilizador/reaction_kingdom.git
   cd reaction_kingdom

2. **Iniciar o servidor**
    docker-compose up -d

3. **Aceder ao jogo**
    Abre o browser em http://localhost:5500

4. **Parar o servidor**
    docker-compose down
